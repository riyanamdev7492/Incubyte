**String Calculator**

## Requirements
- **Ruby Version**: `3.4.1`
- **Bundler**: `2.6.6`
- **RSpec**

## Code Setup

### **Clone the Repository**
```sh
git clone https://github.com/riyanamdev7492/Incubyte.git
cd string_calculator
```

### **Installation**
```sh
bundle install
```

### **Run Test Cases**
```sh
bundle exec rspec
```
